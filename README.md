# KidTask - Task & Wish Management Application 

KidTask is a Java-based desktop application designed to help children manage their daily tasks, track points, and earn wishes with parental approval.

##  Features
- **Task Management:** Add, view, delete, and complete daily tasks.
- **Wish System:** Kids can request rewards (LEGO, Zoo trip, etc.) based on their level.
- **Gamification:** Earn points for tasks, level up, and unlock achievements.
- **Data Persistence:** All data is saved automatically to local files (`Tasks.txt`, `Wishes.txt`).
- **Clean UI:** Modern interface built with Java Swing.

##  Technologies Used
- **Language:** Java
- **GUI:** Java Swing
- **IDE:** IntelliJ IDEA
